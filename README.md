# 卡牌冒险游戏 (Card Adventure Game)

一个结合了Roguelike、卡牌游戏和刮刮卡元素的回合制冒险游戏。

## 游戏特点

- 三种可选角色：战士、法师、游侠
- 多样化的卡牌系统：
  - 普通攻击卡
  - 重击卡
  - 技能卡（雷击、冰冻）
- 刮刮卡系统获取新卡牌
- 增益效果系统
- 回合制战斗系统

## 安装要求

- Python 3.x
- Pygame

## 安装依赖

```bash
pip install pygame
```

## 运行游戏

```bash
python main.py
```

## 游戏玩法

1. 选择角色开始游戏
2. 使用卡牌攻击怪物
3. 通过刮刮卡获取新卡牌
4. 击败怪物后选择增益效果
5. 继续冒险！
